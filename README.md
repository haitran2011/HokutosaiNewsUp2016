## 概要
北斗祭アプリ2016の運営サイドが使用するお知らせ管理アプリです。  
お知らせの閲覧/投稿/削除が行えます。  
北斗祭開幕の2日前に急ぎでスピード開発したものです。（App Storeには提出していません。ログイン機能等も付いていません。）  
北斗祭アプリについては[こちら](https://www.hokutosai.tech)をご覧ください。
